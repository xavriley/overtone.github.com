dpoll

[in :none, trig-id -1, label dpoll-val, run 1.0] in - Demand ugen to poll values from trig-id - If greater than 0, a '/tr' message is sent back to the client (similar to send-trig) label - A label string run - Activation switch 0 or 1 (can be a demand ugen) Print the value of an input demand ugen. The print-out is in the form: label: value block offset: offset. WARNING: Printing values from the Server is intensive for the CPU. Poll should be used for debugging purposes. Categories: Rates: [ :dr ] Default rate: :dr
