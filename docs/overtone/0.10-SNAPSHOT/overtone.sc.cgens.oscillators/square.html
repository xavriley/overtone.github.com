square

A square wave generator [freq 440] freq - Signal frequency A square wave only exists in two states: high and low. This wave produces only odd harmonics resulting in a mellow, hollow sound. This makes it particularly suitable for emulating wind instruments, adding width to strings and pads, or for the creation of deep, wide bass sounds. See the pulse ugen if you wish to modulate the width. Categories: Composite Ugen Rates: [ :ar ] Default rate: :ar
