replace-out:kr

Send signal to a bus, overwriting previous contents. [bus :none, signals :none] bus - The index of the buss to write to. The lowest index numbers are written to the audio hardware. signals - A list of signals or single output to write out. You cannot change the size of this once a synth has been defined. Out adds its output to a given bus, making it available to all nodes later in the node tree (See Synth and Order-of-execution for more information). ReplaceOut overwrites those contents. This can make it useful for processing. Categories: Rates: [ :ar, :kr ] Default rate: :auto
